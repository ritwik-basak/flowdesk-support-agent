# Flowdesk FAQ

This FAQ covers the most common workflow questions for Flowdesk, our project management platform for teams that plan work, track progress, and collaborate across projects. Use the sections below when you need quick guidance for everyday actions inside a workspace.

## How to create a project

Creating a project in Flowdesk gives your team a dedicated space for tasks, timelines, files, and discussion.

1. Sign in to Flowdesk and open the workspace where the project should live.
2. Select **New Project** from the left sidebar or the top navigation bar.
3. Enter a project name that is easy for your team to recognize.
4. Add an optional description to explain the project goal, owner, or delivery date.
5. Choose a starting view such as **Board**, **List**, or **Timeline**.
6. Select a template if you want prebuilt sections for launches, sprints, onboarding, or internal operations.
7. Click **Create Project**.

After the project is created, you can:

- Add tasks and subtasks
- Assign owners and due dates
- Attach files or external links
- Pin project notes for the team
- Set the project status to `Planning`, `Active`, or `On Hold`

If you do not see the option to create a project, your workspace role may be limited. Ask a workspace admin for project creation access.

## How to invite team members to a workspace

You can invite teammates so they can view or contribute to projects in the same workspace.

1. Open the workspace switcher and select the correct workspace.
2. Go to **Settings** > **Members**.
3. Click **Invite Members**.
4. Enter one or more email addresses separated by commas.
5. Choose a role for each user:
- `Admin` can manage billing, workspace settings, and members.
- `Member` can work on projects and tasks they have access to.
- `Guest` can access only shared projects or items they are invited to.
6. Click **Send Invites**.

Invited users receive an email with a secure join link. Pending invites remain visible in the **Members** page until they are accepted, expired, or revoked.

## How to set recurring tasks

Recurring tasks are helpful for weekly reviews, monthly reports, content publishing, and routine operations.

1. Open an existing task or create a new one.
2. In the task details panel, select the **Due Date** field.
3. Turn on **Repeat**.
4. Choose a frequency such as:
- Daily
- Weekly
- Every 2 weeks
- Monthly
- Custom cadence
5. Set an end condition if needed, such as a final date or number of occurrences.
6. Save the task.

When a recurring task is completed, Flowdesk automatically creates the next instance based on the schedule. Comments, task title, assignee, priority, and checklist items can carry forward depending on your recurrence settings.

## How to export a project to CSV

Project export is useful for reporting, stakeholder updates, or backup analysis in spreadsheet tools.

1. Open the project you want to export.
2. Click the three-dot menu in the upper-right corner.
3. Select **Export** > **CSV**.
4. Choose whether to export:
- All tasks
- Only active tasks
- A filtered view
- Completed tasks only
5. Confirm the export.

The CSV file typically includes task name, assignee, due date, status, priority, labels, and created date. Attachments, comments, and rich formatting are not included in CSV exports.

## How to use keyboard shortcuts

Flowdesk includes keyboard shortcuts to help frequent users move faster.

To open the shortcut guide:

- Press `?` anywhere in the web app

Common shortcuts include:

- `C` to create a new task
- `P` to open project search
- `G` then `H` to go to the Home dashboard
- `G` then `M` to go to My Tasks
- `/` to focus the search bar
- `E` to edit the selected task
- `Ctrl + Enter` or `Cmd + Enter` to save changes

Shortcuts may be disabled while typing in text fields. Some combinations vary slightly by browser and operating system.

## How to set task priorities

Task priorities help teams decide what needs immediate attention.

To change priority:

1. Open the task card or task details drawer.
2. Find the **Priority** field.
3. Choose one of the standard values:
- `Low`
- `Medium`
- `High`
- `Urgent`
4. Save the task if autosave is not enabled in your workspace.

You can also sort and filter project views by priority to surface critical work first.

## How to archive a project

Archiving is recommended when a project is complete but you still want to preserve its history.

1. Open the project.
2. Click the three-dot menu.
3. Select **Project Settings**.
4. Choose **Archive Project**.
5. Confirm the action.

Archived projects are removed from active navigation and reporting views, but authorized users can still restore them later from the archived projects list. Archiving does not delete tasks, files, comments, or activity history.

## How to change notification settings

Flowdesk lets each user control how and when they receive updates.

1. Click your profile avatar.
2. Go to **My Settings** > **Notifications**.
3. Review the available notification categories:
- Task assignments
- Due date reminders
- Comment mentions
- Status changes
- Daily digest emails
- Mobile push notifications
4. Turn categories on or off based on your preference.
5. Save your changes.

Workspace admins can also define default notification rules for new members, but personal settings usually override those defaults. If you stop receiving expected updates, make sure browser notifications are allowed and that your email address is verified.
